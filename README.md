# Проект: Место

## Обзор

Сайт-однстраничник с возможностью редактирования контента и загрузки изображений на сервер.

## Использованные технологии:

* HTML5
* CSS3 (BEM, Flexbox, Grid Layout, Adaptive UI)
* JavaScript
* Webpack

## Обновлено:

* возможность загрузки карточек на сервер
* возможность редактирвания аватара
